ROS lib, camera stream node
ROS library for image transmission.

This template for a ROS library was intended to be implemented in the Rescue Robot design by us for Tec de Monterrey, Campus Guadalajara.
The same library contain both nodes (transmit and recevie), but it is not necessary to include the receive node in the robot (this could avoid unnecessary use of memory).
